Why this page exists

The site already had a browser-formatted HTML report and compact capability maps. The new report pack adds a separate source-driven bilingual narrative in Markdown and PDF.

Evidence boundary

Both reports were written from direct inspection of the committed source tree. The user-supplied PDFs were treated as outside reference, not as source evidence.

Regeneration path

The PDFs are produced from the Markdown sources by scripts/render_markdown_pdf.py, so future edits can keep the artifact chain reproducible.
