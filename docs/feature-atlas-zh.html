Claude Code 源码能力地图

Claude Code Source Capability Map

这页只基于当前提交进仓库的 src/ snapshot 来写。它把源码中能直接看到的能力面， 按“广泛实现的主干”“明确存在但带 gate 或范围约束的能力族”“当前只能部分验证的入口或子系统”分层整理， 不把外部发布状态、来源故事或营销式说法混进技术判断。 This page is derived only from the committed snapshot. It separates broadly implemented runtime spine, specialized but gated surfaces, and snapshot-limited evidence without making release-status or provenance claims.

  • 运行时主干
    持久化、模型循环、工具执行、远端会话和交互 UI 都是深实现。
  • 专门能力家族
    很多显眼功能确实存在，但常常受 gate、平台或 entitlement 约束。
  • 证据边界

证据规则

实装、受控、部分可见 / implemented, gated, partial

“实装”表示存在可执行代码或已接上的 UI / service 路径；“受控”表示能力真实存在但受 flags、平台或 entitlement 限制； “部分可见”表示只有入口或局部，缺少完整子系统可供检查。 Implemented means wired code; gated means real but constrained; partial means only part of the subsystem is present.
